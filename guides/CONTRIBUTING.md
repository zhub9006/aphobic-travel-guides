# 🤝 Contributing to Bug-Free Travel Guides

Thank you for wanting to help build the most comprehensive bug-free travel resource on the internet!

## How to Add a New Destination

1. **Open a GitHub Issue** with title: `New Destination: [Place Name]`
2. Include the following details:
   - **Location:** Country, region, and specific city/area
   - **Your Bug Experience:** What insects did you encounter (or did not)?
   - **Best Season:** When is insect activity lowest? When is it worst?
   - **Why It's Bug-Free:** Climate, altitude, wind, resort treatment, government programs?
   - **Caveats:** Any exceptions, seasonal changes, or areas to avoid within the destination
   - **Sources:** Links to travel blogs, tourism boards, entomology reports, or government advisories
3. We will review your submission, format it to match the guide style, and add it to the next update!

## How to Improve Existing Guides
- **Outdated info?** Open a PR with the correction and your source.
- **New tips?** Add them to the appropriate section.
- **Bug in research?** (pun intended) Open an issue with evidence.

## Style Guidelines
- Use the table format for destination details (see `guides/destination-guide-1-research-verified-bug-free-travel.md`)
- Include seasonal notes with specific months
- Cite at least one source per destination
- Use the rating system: Zero = 0 bugs, Very Low = 1-2 insects rarely, Low = minimal precautions needed
- Be honest about caveats — no destination is perfect!
